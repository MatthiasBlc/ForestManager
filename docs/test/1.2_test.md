Manual Testing Checklist

You can now test the following scenarios at  
 http://localhost:3000:

Login (Modal)

1. Click "Log In" in NavBar - modal opens
2. Enter valid credentials - modal closes, user logged in
3. Enter invalid credentials - error AUTH_008 displayed
4. Click outside modal - modal closes  


Signup (Page)

5. Navigate to /signup - dedicated signup page
6. Test validation errors (short username, invalid email, weak
   password)
7. Submit valid signup - redirects to / logged in
8. Visit /signup while logged in - redirects to /  


Routes & Session

9. Visit / while logged out - redirects to /signup
10. Refresh page while logged in - session restored
11. Click "Log out" - user logged out, can see logged out view
